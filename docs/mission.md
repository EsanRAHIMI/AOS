# Mission

Deliver a clean, scalable, multi-agent kernel where:
- Goals enter through the gateway and are decomposed by the orchestrator.
- Specialist agents (architect, builder, devops, memory, …) execute traceable work.
- Every action is logged, streamed live, and persisted to MongoDB Atlas.
- Artifacts live in AWS S3 with metadata tracked in MongoDB.
- The system asks for approval before anything sensitive or irreversible.
- Documentation and memory are updated after every meaningful phase.

Success = a foundation strong enough that future agents can extend the system
with minimum confusion and minimum token usage.
